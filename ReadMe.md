# GSJPプロファイルダウンロードツール

## 概要
このバッチファイルは、Microsoft Flight Simulator用のGSX（Ground Services X）プロファイルを、GitHubのGroundServicesJPリポジトリから自動的にダウンロード・インストールするツールです。

## 必要な環境
- **GSX Pro for MSFS**がインストールされていること
- **Git**がインストールされていること
  - Gitのインストール方法：[https://git-scm.com/downloads](https://git-scm.com/downloads) からダウンロードしてインストール
  - インストール時は基本的にデフォルト設定のまま「Next」で進めてOKです

## インストール方法

### 1. バッチファイルの保存
1. このバッチファイルを任意の場所に保存します
2. **保存場所の例**：
   - デスクトップ
   - ドキュメントフォルダ
   - 専用のツールフォルダ（例：`C:\FlightSimTools\`）

> **注意**：保存場所は **GSXのインストール先でなくてOK** です。どこに置いても動作します。

### 2. 実行方法
1. 保存したバッチファイルを**ダブルクリック**して実行
2. 内容を確認し、問題なければ**Enterキー**を押す
3. ダウンロードとインストールが自動的に実行されます
4. 「完了しました」と表示されたらウィンドウを閉じてください

## ⚠️ 重要な注意事項

### バックアップについて
**gsjpのファイルを編集している場合は、必ず事前にバックアップしてください：**

このツールを実行すると、`%APPDATA%\Virtuali\GSX\MSFS\` フォルダ内のgsjpさんで配布されているファイルが上書きされます。バックアップを取るか、編集したファイルの名前を変更してください。

### 削除されるファイル（デフォルト設定の場合）
以下のファイルは自動的に削除されます：
- **熊本空港（RJFT）snjsim版**
  - `rjft-snjsim.py`
  - `rjft-snjsim.ini`
- **利尻空港（RJER）Gate15 v2版**
  - `rjer-g15v2.py`
  - `rjer-g15v2.ini`

### 削除されないファイル
- 熊本空港 keisim版（`rjft-keisim.py/.ini`）
- 利尻空港 Marin版・Gate15 v1版（`rjer-g15_marin.py/.ini`）
- gsjp配下以外のプロファイル

## カスタマイズ方法

### 使用しているシーナリーに合わせた設定
上記デフォルトで削除されるファイルを使用している場合、バッチファイルを編集して削除対象を変更できます。

#### 編集手順
1. バッチファイルを**右クリック** → **編集**（またはメモ帳で開く）
2. 以下の行を探して修正します：

**熊本空港でsnjsim版を使っている場合：**
```bat
rem snjsim版お使いの際は、snjsim → keisim に変更してください
del rjft-snjsim.py 2>nul
del rjft-snjsim.ini 2>nul
```
↓ 以下のように変更
```bat
rem snjsim版お使いの際は、snjsim → keisim に変更してください
del rjft-keisim.py 2>nul
del rjft-keisim.ini 2>nul
```

**利尻空港でGate15 v2版を使っている場合：**
```bat
rem Gate15_v2版お使いの際は、g15v2 → g15_marin に変更してください
del rjer-g15v2.py 2>nul
del rjer-g15v2.ini 2>nul
```
↓ 以下のように変更
```bat
rem Gate15_v2版お使いの際は、g15v2 → g15_marin に変更してください
del rjer-g15_marin.py 2>nul
del rjer-g15_marin.ini 2>nul
```

3. ファイルを**上書き保存**して閉じる

## トラブルシューティング

### 「'git' は、内部コマンドまたは外部コマンド...」というエラーが出る
- **原因**：Gitがインストールされていない、またはパスが通っていない
- **解決策**：Gitをインストールしてください（上記「必要な環境」参照）

### 「処理中にエラーが発生しました」と表示される
- **原因1**：インターネット接続の問題
  - GitHubに接続できることを確認してください
- **原因2**：GSXが正しくインストールされていない
  - GSX Pro for MSFSが正しくインストールされているか確認してください
- **原因3**：ファイルアクセス権限の問題
  - バッチファイルを**管理者として実行**してみてください（右クリック→管理者として実行）

## ファイルの配置先
ダウンロードされたプロファイルは以下の場所に配置されます：
```
%APPDATA%\Virtuali\GSX\MSFS\
```

実際のパスの例：
```
C:\Users\(ユーザー名)\AppData\Roaming\Virtuali\GSX\MSFS\
```
